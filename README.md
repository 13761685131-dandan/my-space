# My Space

个人成长数字花园 · grow a little every day

## 功能

- 💭 **情绪追踪** — 每天记录心情，日历热力图
- 📚 **学习志** — 读书、播客、让你有领悟的一切
- ✈️ **旅行志** · 👗 **穿搭志** · 🍜 **美食志** · 🏆 **职场志**
- 📸 **照片上传** — 所有模块都支持图片
- 🖼️ **分享导出** — 日签（随性拼贴）· 拼图 · 情绪卡片 · 月度画报
- 📱 **PWA** — 添加到主屏幕，离线可用

## 使用

打开 `index.html` 或用任意静态服务器：

```
python3 -m http.server 8080
```

数据全部存储在浏览器 localStorage，不会上传到任何服务器。
